# Roadmap RetroNet 8080

## v0.1

- core CPU importabile
- decoder completo a 256 opcode
- memoria 64 KB e I/O 256 porte
- famiglie istruzionali principali 8080
- disassembler e trace CLI
- profili conservativi
- terminale callback
- front panel, READY/WAIT e interrupt esterno
- debugger e trace JSON
- conformance sintetica

## Dopo v0.1

- confronto differenziale con emulatori indipendenti
- backend `i8080` in `retronet-asm` con esempi stabili (completato)
- mappe storiche Altair/IMSAI solo da fonti primarie
- integrazione con `retronet-cpm` (completata nel repo CP/M-like)
- supporto a formati immagine e periferiche documentate
