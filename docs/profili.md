# Profili Macchina

I profili descrivono una macchina sopra il core senza inventare dettagli storici.

- `generic`: RAM piatta da 64 KB.
- `altair-8800`: profilo conservativo per Altair 8800, senza ROM incluse.
- `imsai-8080`: profilo conservativo per IMSAI 8080, senza mappe S-100 inventate.
- `cpm-dev`: profilo didattico non storico per futuri esperimenti CP/M-like.

Gli slot ROM `monitor` e `test` accettano file locali dell'utente. Un hash o una
corrispondenza funzionale non autorizzano la redistribuzione di ROM storiche.
